# Go Student Service

A Go service that generates PDF reports from student data payload.

## Features

- ✅ Generate PDF reports directly from student data payload
- ✅ RESTful API endpoints  
- ✅ Health check endpoint
- ✅ Clean, modular architecture
- ✅ No external API dependencies

## API Endpoints

- `GET /health` - Health check endpoint
- `POST /students/pdf` - Generate PDF report from student data in request payload

## Quick Start

1. **Install dependencies:**
   ```bash
   go mod tidy
   ```

2. **Run the service:**
   ```bash
   go run .
   ```

3. **Test the endpoints:**
   ```bash
   # Health check
   curl http://localhost:8080/health
   
   # Generate PDF from payload data
   curl -X POST http://localhost:8080/students/pdf \
     -H "Content-Type: application/json" \
     -d '{

       "name": "John Smith",
   ```

## Architecture

```
├── main.go              # Application entry point
├── demo-payload-pdf.sh  # Demo script
├── config/
│   └── config.go        # Configuration management
├── handlers/
│   └── handlers.go      # HTTP handlers
├── pdf/
│   └── pdf.go          # PDF generation logic  
├── student/
│   └── student.go      # Student data structure
└── go.mod              # Go module definition
```

## PDF Report Includes

- **Basic Information**: Name, ID, Email, Phone, Gender, DOB
- **Academic Information**: Class, Section, Roll, Admission Date, System Access
- **Family Information**: Father, Mother, Guardian details with contact info
- **Address Information**: Current and Permanent addresses
- **Report Metadata**: Generated by, timestamp

## Dependencies

- `gorilla/mux` - HTTP router and URL matcher
- `jung-kurt/gofpdf` - PDF generation library

## Example Usage

```go
// Initialize the PDF generator
pdfGenerator := pdf.NewGenerator()

// Create server
server := handlers.NewServer(pdfGenerator)

// Student data from request payload
var student student.Student
json.Unmarshal(requestBody, &student)

// Generate PDF
pdfBytes, err := pdfGenerator.GenerateStudentReport(&student)
```

## Docker Support

Create a `Dockerfile`:

```dockerfile
FROM golang:1.21-alpine AS builder
WORKDIR /app
COPY . .
RUN go mod tidy && go build -o student-service

FROM alpine:latest
RUN apk --no-cache add ca-certificates
WORKDIR /root/
COPY --from=builder /app/student-service .
CMD ["./student-service"]
```

Build and run:
```bash
docker build -t student-service .
docker run -p 8080:8080 -e STUDENT_API_URL=https://your-api.com student-service
```
